3. Technical Architecture The project is built on a robust Client-Server Architecture:

Frontend: A modern, responsive User Interface developed with HTML5, Tailwind CSS, and JavaScript. It handles file inputs and presents complex analysis data in an intuitive, graphical format.

Backend: A Python (Flask) server acts as the core engine, managing API endpoints and orchestrating the heavy AI processing tasks via TensorFlow and PyTorch.

Security: We utilize werkzeug for secure file handling and have enabled Cross-Origin Resource Sharing (CORS) to ensure safe communication between the client and server.

4. Innovation & Impact This tool goes beyond simple detection; it is a complete forensic utility. The "Forensic Report Generation" feature provides users with a downloadable PDF report detailing the analysis, which is crucial for professional verification. By combining deep learning (fine-tuned XceptionNet) with heuristic analysis, we ensure a robust and reliable detection mechanism.
